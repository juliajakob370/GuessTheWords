Welcome to the **Guess the Words Game**!

**HOW TO PLAY:**

  • Press START to begin the game
  • You will have 3 minutes to play
  • Find hidden words in the 30-character string
  • Words can appear forward or backward
  • Type a word and press GUESS to submit.
  • You will get feedback based on your guess: Found / Not Found / Already Submitted.

This game demonstrates the use of TCP-IP and can be played on multiple computers by configuring IP and Port numbers in the App.Config!
It uses a Server-Client model!
